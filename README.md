# bangalore-house-price-prediction
**Problem Statement**
Built an end-to-end machine learning solution to predict house prices in Bangalore based on features such as location, square footage, number of bedrooms (BHK), and number of bathrooms.

**Tech Stack**
Python
Pandas & NumPy
Scikit-learn
Flask (Backend API)
HTML/CSS/JavaScript (Frontend)
AWS EC2 (Deployment)
Nginx (Reverse Proxy)

** Machine Learning Approach**
Data Cleaning & Preprocessing
Outlier Detection & Removal
Feature Engineering
One-Hot Encoding (Location Feature)
